# Security Policy

## Supported Status

This is an early public prototype. Do not store private nonprofit data, API secrets, or restricted datasets in the repository.

## Reporting Issues

Open a GitHub issue for non-sensitive security concerns. For sensitive reports, contact the repository owner privately through GitHub.

## Data Rules

- Keep API keys in environment variables.
- Do not commit `.env.local`, Neon connection strings, raw private datasets, or paid-source payloads.
- Public evidence should respect source licenses and redistribution terms.
- AI-generated claims should remain draft until reviewed.
